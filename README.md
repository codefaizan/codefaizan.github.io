# Faizan Ahmad - Portfolio Website

A modern, animated portfolio website showcasing mobile and backend engineering expertise.

## 🎨 Features

- **Modern Design**: Sleek dark theme with gradient accents and smooth animations
- **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- **Interactive Elements**: 
  - Smooth scroll navigation
  - Animated floating cards
  - Typing effect for subtitle
  - Parallax scrolling effects
  - 3D tilt effect on project cards
  - Cursor trail effect (desktop only)
  - Animated counters for statistics
  - Scroll-to-top button
- **Sections Included**:
  - Hero section with call-to-action
  - About me with professional summary
  - Work experience timeline
  - Featured projects showcase
  - Skills & technologies grid
  - Education & certifications
  - Contact form with social links

## 🚀 Technologies Used

- CSS3 (Flexbox, Grid, Animations)

## 🔧 Customization

To customize this portfolio:

1. **Update Personal Info**: Edit the content in `index.html`
2. **Change Colors**: Modify CSS variables in `:root` section of `style.css`
3. **Add/Remove Sections**: Adjust HTML structure and update navigation
4. **Modify Animations**: Edit animation keyframes in `style.css` and JavaScript in `script.js`
